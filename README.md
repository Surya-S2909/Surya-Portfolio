# Surya S — Developer Portfolio

A personal portfolio website showcasing my skills, projects, and background as a Java Full Stack Developer. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step.

🔗 **Live demo:** _add your deployed link here (GitHub Pages / Netlify / Vercel)_

## Preview

A dark, developer-themed portfolio with a code-editor-style hero section, 3D tilt effects on cards, floating tech badges, and scroll-triggered animations.

## Features

- 🖥️ Code-editor-inspired hero section with a typed "about" snippet
- 🎯 3D tilt interactions on the profile photo, project cards, and skill cards
- 🪄 Scroll-reveal animations as you move down the page
- 📱 Fully responsive — works on mobile, tablet, and desktop
- ♿ Respects `prefers-reduced-motion` for accessibility
- 🔗 Linked GitHub, LinkedIn, email, and phone contact options

## Sections

- **Hero** — introduction with photo and quick stack overview
- **About** — summary and quick facts
- **Skills** — grouped by Languages, Frontend, Backend & Database, Authentication, Tools, and Concepts
- **Projects** — featured project cards linking out to GitHub repos
- **Education & Certificates** — academic background and certifications
- **Contact** — direct links to reach out

## Tech stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties, Grid, Flexbox, animations) |
| Interactivity | Vanilla JavaScript (Intersection Observer, mouse-tracked 3D tilt) |
| Fonts | Space Grotesk, Inter, JetBrains Mono (Google Fonts) |

## Getting started

Clone the repo and open `index.html` directly in your browser — no installation or build tools required.

```bash
git clone https://github.com/Surya-S2909/portfolio.git
cd portfolio
```

Then just open `index.html` in your browser, or serve it locally:

```bash
# using Python
python3 -m http.server 8000
```

Visit `http://localhost:8000` in your browser.

> **Note:** Keep `index.html` and `surya_img.jpeg` in the same folder — the photo is referenced with a relative path.
